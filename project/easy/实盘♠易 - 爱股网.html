功能

实盘易是管理通达信、同花顺等交易客户端，并对为A股量化投资者提供基于 HTTP 协议 RESTFul API 的软件。

  • 交易
    • 买入、市价买入
    • 卖出、市价卖出
    • 撤单
  • 查询
    • 资金股份
    • 当日委托
    • 当日成交
    • 当日未成交
    • 其他自定义查询
  • 其他
    • RESTful API
    • 支持多账户
    • 支持融资融券账户
    • 自动登录
    • 自动打新
    • 多平台（聚宽、米筐、MT4等）集成

安装

最低配置

  • Windows 7 及以上操作系统，Windows Server 2008 及以上操作系统
  • 内存 1GB 以上

推荐配置

  • Windows Server 2012 R2
  • 内存 1GB 以上

本地机器

实盘易在下单过程中会弹出窗口。为确保交易不被干扰，建议您在专用机器或虚拟机中部署实盘易。

云服务器

在云服务器（如阿里云等）上部署实盘易时，由于 Windows 自带的远程桌面连接（RDP）存在一些技术问题，因此必须使用 VNC。这里我们推荐 TightVNC。

API 文档

见主界面 帮助>查看帮助 菜单。 

查询账号
GET http://localhost:8888/accounts

查询资金股份
GET http://localhost:8888/positions

查询投资组合
GET http://localhost:8888/positions
Accept: application/vnd.joinquant+json

查询当日委托
GET http://localhost:8888/orders

查询撤单
GET http://localhost:8888/orders?status=open

查询当日成交
GET http://localhost:8888/orders?status=filled

买入
POST http://localhost:8888/orders
Content-Type: application/json

{
  "action": "BUY",
  "symbol" : "000001",
  "type": "LIMIT",
  "priceType" : 0,
  "price" : 9.00,
  "amount" : 100
}

type 可选择：

LIMIT - 限价单，需指定 price
MARKET - 市价单，需指定 priceType（非 0）
priceType 可选择：
上海交易所：

0 - 限价委托
4 - 五档即时成交剩余撤销
6 - 五档即时成交剩余转限
深圳交易所：

0 - 限价委托
1 - 对手方最优价格委托
2 - 本方最优价格委托
3 - 即时成交剩余撤销委托
4 - 五档即时成交剩余撤销
5 - 全额成交或撤销委托
amountProportion 和 amount 二选一
amountProportion 可选择：ALL, 1/2, 1/3, 1/4, 1/5

以上参数设置也适用于"卖出"

卖出
POST http://localhost:8888/orders
Content-Type: application/json

{
  "action": "SELL",
  "symbol" : "000001",
  "type": "LIMIT",
  "priceType" : 0,
  "price" : 9.00,
  "amount" : 100
}

撤单
DELETE http://localhost:8888/orders/O1234
（注：上面 URL 中的 O1234 应替换为相应的委托编号）

撤单全部
DELETE http://localhost:8888/orders

查询其他
GET http://localhost:8888?navigation=查询>当日委托

查询状态
GET http://localhost:8888/statuses

生成调仓信号
POST http://localhost:8888/adjustments
(详见用户手册)

查询交易客户端状态
GET http://localhost:8888/clients
GET http://localhost:8888/clients/client-1

更改交易客户端状态
PATCH http://localhost:8888/clients
PATCH http://localhost:8888/clients/client-1
       {
              "status": "LOGGED"
       }

(详见用户手册)

多账户

多交易客户端的情况下，可以在 URL 中添加 client 参数以区分交易客户端。如果系统中只运行了一个交易客户端，可忽略该参数。

client 参数值可以是某一交易客户端标题关键字、交易账号关键字的汉字、拼音。

规则：client=[title|account|*]:<关键字>[,...]

以“通达信模拟交易终端”，登录交易账号为“15012345678”为例，查询账号可以选用以下任一 URL。

  • http://localhost:8888/accounts?client=title:monijiaoyi
  • http://localhost:8888/accounts?client=title:模拟交易 （“模拟交易”需做 URL encoding）
  • http://localhost:8888/accounts?client=account:150
  • http://localhost:8888/accounts?client=*:monijiaoyi
  • http://localhost:8888/accounts?client=*:150

安全

如需做权限设置，请在URL中添加 key 参数。key 值的设置请参阅“设置”章节。

规则：key=[配置中定义的值]

介绍

流程

  • 联系爱股网站长报名参加活动。

  • 合作券商负责人按月通过爱股网转账发放相应积分。

该活动最终解释权归爱股网所有。
